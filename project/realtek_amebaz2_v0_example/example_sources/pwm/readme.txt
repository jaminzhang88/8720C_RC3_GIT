Example Description

This example describes how to use PWM.

Requirement Components:
    1~4 LED

Connect LED to below PWM pins and GND, then the LED will gradually become brighter and then darker with different speed.
    1, connect PA_1 to a LED and GND in series
    2, connect PA_2 to a LED and GND in series
    3, connect PA_3 to a LED and GND in series
    4, connect PA_4 to a LED and GND in series
